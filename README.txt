Integrates with various web apps to import statuses as posts

== Description ==

This plugin is intended to create a public repository of status updates from various web apps. I've started with Twitter, and will be uploading new integrations as time passes.

== Installation ==

1. Upload `david-vg.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the Settings page for the plugin to change all relevant settings to import accounts

== Frequently Asked Questions ==

= What integrations does the plugin have? =

Current integrations include:
* Twitter

= Can information be stored privately? =

A setting will be added to flag specific channels as private. Still, it'll only be as secure as your site is.
